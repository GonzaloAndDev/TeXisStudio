# TeXisStudio — Community Contributions

This directory holds community-contributed resources that extend TeXisStudio
without modifying core app code.

| Directory     | Contents                                        |
|---------------|-------------------------------------------------|
| `dictionaries/` | Hunspell spell-check dictionaries (`.aff` + `.dic`) |
| `locales/`    | UI translations (`translation.json` per locale) |

## Quick links

- [How to contribute a dictionary](dictionaries/README.md)
- [How to contribute a translation](locales/README.md)
- [Main repository](https://github.com/GonzaloAndDev/TeXisStudio)

## In-app access

Users can browse and install community packages directly from
**Settings → Community packages** inside TeXisStudio.
