## Before you start

TeXisStudio needs a LaTeX backend. Install **Tectonic**, which downloads resources on demand, or a complete suite with **latexmk**, such as MacTeX, TeX Live, or MiKTeX. The setup screen detects your operating system and shows specific instructions.

## Create a project

1. Select **New project** on the home screen.
2. Choose a document type and institutional profile.
3. Complete the requested metadata and select a folder.
4. Review the summary before creating the project.

## Understand the editor

The left column contains sections. The center shows blocks for the active section. The right panel contains project information and options when space allows; narrow windows hide it to preserve writing space.

Use the element toolbar to insert paragraphs, headings, lists, equations, figures, and other blocks. You can also open the palette with `Ctrl+K` or `⌘K`.

## Save and compile

The app saves changes automatically. `Ctrl+S` or `⌘S` forces an immediate save.

Select **Compile** in the top bar. The compilation screen lets you review the backend, progress, diagnostics, and generated PDF.

:::caution
The current version has no global keyboard shortcut for **Compile**. Use the visible top-bar button.
:::
