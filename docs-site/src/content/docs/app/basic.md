## Antes de empezar

TeXisStudio necesita un backend LaTeX. Puedes instalar **Tectonic**, que descarga recursos según los necesita, o una suite completa con **latexmk** como MacTeX, TeX Live o MiKTeX. La pantalla de instalación detecta tu sistema y muestra instrucciones específicas.

## Crear un proyecto

1. En Inicio, selecciona **Nuevo proyecto**.
2. Elige tipo de documento y perfil institucional.
3. Completa los metadatos solicitados y selecciona la carpeta.
4. Revisa el resumen antes de crear el proyecto.

## Entender el editor

La columna izquierda contiene las secciones. El centro muestra los bloques de la sección activa. El panel derecho contiene información y opciones del proyecto cuando hay espacio suficiente; en ventanas estrechas se oculta para conservar el área de escritura.

Usa la barra de elementos para insertar párrafos, títulos, listas, ecuaciones, figuras y otros bloques. También puedes abrir la paleta con `Ctrl+K` o `⌘K`.

## Guardar y compilar

La app guarda los cambios automáticamente. `Ctrl+S` o `⌘S` fuerza un guardado inmediato.

Selecciona **Compilar** en la barra superior. Se abrirá la pantalla de compilación, donde puedes revisar el backend, el progreso, los diagnósticos y el PDF generado.

:::caution
La versión actual no asigna un atajo global a **Compilar**. Usa el botón visible en la barra superior.
:::
